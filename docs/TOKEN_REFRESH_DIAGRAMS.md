## Key Points

1. **Automatic Refresh**: Happens 1 minute before token expires
2. **Seamless**: User never notices the refresh happening
3. **Fallback**: If automatic refresh fails, 401 errors trigger manual refresh
4. **Security**: Tokens rotate on each refresh
5. **Cleanup**: Logout properly clears everything
